Project Overview

Agency Measures Tracker is an automated reporting system that syncs CSV data, processes agency metrics, and generates weekly performance reports. Built with Google Apps Script and JavaScript, it streamlines manual workflows and ensures timely, accurate reporting.

Key Features

  • Automated CSV data sync
  • Metric processing and validation
  • Weekly report generation
  • Customizable agency KPIs
  • Integration with Google Sheets

Impact & Results

Reduced manual reporting time by 75%, improved data accuracy, and enabled agencies to focus on analysis rather than data wrangling. The system is used by multiple agencies for streamlined performance tracking and reporting.
