Project Overview

Agency Metric Tracker is an automated reporting system that syncs CSV data, processes agency metrics, and generates weekly performance reports. Built with Google Apps Script and JavaScript, it streamlines manual workflows and ensures timely, accurate reporting.

Key Features

  • Automated data import from multiple CSV sources
  • Real-time metric validation with error detection
  • Scheduled weekly reports delivered automatically
  • Customizable KPI dashboards tailored to agency needs
  • Seamless Google Sheets integration for familiar workflows

Impact & Results

Reduced manual reporting time by 75%, eliminating up to 10 hours of weekly data processing. Improved data accuracy through automated validation and enabled agency teams to focus on strategic analysis rather than manual data management. Currently deployed across multiple agencies, processing thousands of data points quarterly.
